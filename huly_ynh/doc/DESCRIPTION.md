# Huly Selfhost

Huly is an open-source alternative to Linear and Jira with integrated Chat and Video.

It features:
- Issue Tracking
- Chat and Collaboration
- Extensive internal apps

**Note:** This is an incredibly heavyweight application using 14 distinct docker containers including Redpanda, CockroachDB, and Elasticsearch. You are strongly advised to run this on a host with at least 8 GB of RAM.
